Release History

v1.0.4

  • Fixed the Leave a Review link

v1.0.3

  • Fixed issue with owner change notifications when an extension is updated
  • Improved error handling

v1.0.2

  • Fixed bug preventing extension updates being logged and notified

v1.0.1

  • Added link to options page from popup
  • Added a minimum threshold option for review warnings (defaulted to 5)
  • Added option to remove an uninstalled extension and it's history
  • No longer log "Disabled" event in history when an extension is uninstalled
  • Add badge text, notification and show about tab in popup whenever Invigilator is updated to a new version.

  • Initial release. What could possibly go wrong?

    • View previously uninstalled extensions

    • Alerts when extensions are updated
